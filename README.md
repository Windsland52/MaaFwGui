# MaaFwGui

## install deps

```powershell
$Env:ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
npm install
```

```shell
export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
npm install
```

## dev watch

```shell
node scripts/watch.mjs
```

## build

```shell
node scripts/build.mjs
npm run package
```
